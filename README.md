# 🗺️ Inteligência de Reversa

Visualização interativa de pontos de localização com **VisQuill GDK** + **React** + **TypeScript**.

## ✨ Características

- ✅ Mapa interativo com VisQuill GDK
- ✅ Adicionar/visualizar pontos dinamicamente
- ✅ Interface moderna e responsiva
- ✅ Hover e click para detalhes
- ✅ Deploy pronto para Render.com
- ✅ 8 pontos de exemplo inclusos
- ✅ Integração com API externa

## 🚀 Quick Start

### 1. Clone e instale
```bash
git clone https://github.com/eduardoempilha/Inteligencia-de-Reversa.git
cd Inteligencia-de-Reversa
npm install
```

### 2. Execute localmente
```bash
npm run dev
# Abra http://localhost:3000
```

### 3. Deploy no Render.com
- Vá para [render.com](https://render.com)
- Conecte este repositório
- Deploy automático!

## 📦 Stack

- React 18 + TypeScript
- VisQuill GDK (visualização vetorial)
- Vite (build tool)
- CSS3 responsivo

## 🎯 Uso

1. **Ver pontos**: Mapa com 8 locais predefinidos
2. **Adicionar**: Click "➕ Novo Ponto", preencha dados
3. **Detalhes**: Click em qualquer ponto
4. **Recarregar**: Click "🔄 Recarregar"

## 🌐 Deploy

Render.yaml está configurado. Simplesmente faça push para `main`:
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

Render.com detectará automaticamente e fará deploy!
